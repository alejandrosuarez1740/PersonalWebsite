Automated Metadata tagging and Content Summarization

This was my Senior Design Process in order to graduate from College. We were given the task to create a Metadata Tagging and content Summarization when a PDF, WORD or Video were uploaded.

Metadata describes learning content and will include a short metadata tag of several words, a longer description of the subject using T5 abstractive text summarization, the page and line number and the document it was found in.

This information will be stored in a database. The focus is on documents and videos. Video files uploaded to the system will generate a transcript through a sub-system before running through the main system.

Solution will also generate a Table of Contents, a document and video summary. Example videos and documents related to ship handling.

A front end will also be created to access the database to search and access tagged content.
